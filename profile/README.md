Built by **Sigma**. Backed by purpose. Driven by community.

## 🧬 What is Alsania?

**Alsania** is a modular, ethical, and self-sovereign Web3 ecosystem — designed for creators, developers, and users who want to **own their identity**, **launch their ideas**, and **escape the rent-seeking giants** of Web2.

Everything in Alsania is:
- 💡 **Modular** (Upgradeable, swappable, open)
- 💸 **Low-cost** (Gas-optimized, free to use, no forced subscriptions)
- 🧠 **AI-aware** (Persistent memory, agent support, traceable data)
- 🧱 **Stack-native** (No vendor lock-in)

## 🛠️ Core Projects

### 🏷️ Alsania Enhanced Domains (AED)

> Your decentralized identity and on-chain namespace.

- Mint `.alsania`, `.fx`, `.aed`, `.alsa`, `.07`, '.echo' domains
- Enhance with subdomain minting, profiles, reverse records
- Upgrade existing ENS/Unstoppable domains
- SVG tokenURI visuals, admin dashboard, monetized enhancements

✅ **Live on Polygon Amoy**

---

### 🛒 AlsaniaFX Marketplace

> NFT & token marketplace, sovereign by design.

- ERC721/1155 support
- Fixed-price + auction models
- Creator royalties + ERC20 whitelist (e.g., ALSC)
- Insanely low platform fees. just enough to keep FX running, the test belongs to the creators and Alsania DAO

---

### ⛓️ AlsaniaChain (Mainnet via Polygon CDK)

> Alsania’s own L2 EVM chain

- Built with `cdk-opgeth` in sovereign mode
- Testnet: **Aelion**
- Gas token: **ALSC**
- Protocol-level royalties
- AED and AlsaniaFX integration

---

### 🧠 AlsaniaMCP (Modular Control Plane)

> Local memory, agent orchestration, persona locking

- SSE + REST API
- Postgres + Qdrant + IPFS snapshotting
- Multi-agent support (Echo, Cypher)
- BLAKE3-secured memory ID + snapshot forensics

---

### 💻 DevConX ( IDE Plugin )

- Fork of Continue.dev, powered by AlsaniaMCP
- Project memory, persona persistence, chaos safeguards
- No API key required

---

### 🪙 ALSC Token

> Native gas & utility token of Alsania

- UUPS upgradeable ERC20
- Staking hooks ready
- Used for gas, rewards, and fees
